# FoodiePlan
A web application to plan your meals and learn recipes. 
Learning MERN stack with this project.

## Tech stacks: 
- FrontEnd
    - React
    - Tailwind CSS

- BackEnd
    - Express.js
    - APIs
        - RESTful API
        - GraphQL API
    
    - Database
        - MySQL
        - MongoDB
        - Prisma ORM

- Deployment
    - GitHub Actions
    - Vercel

## Database Design
- Recipe:
    - ID
    - Name
    - Description
    - Ingredients
    - Total Price

- Ingredient
    - ID
    - Name
    - Amount
    - Price
    - Recipe ID
